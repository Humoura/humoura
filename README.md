# Humoura

See [humoura/README.md](./humoura/README.md) for full documentation.

🌐 **Live Demo:** [humoura.vercel.app](https://humoura.vercel.app)

## Quick Start

```bash
cd humoura
npm install && npm start

# In another terminal:
cd humoura/frontend
npm install && npm run dev
```

Open **http://localhost:5173** and click any demo user to log in instantly.
